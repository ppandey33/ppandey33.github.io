2. Zero Initialization

While initializing neural network weights to zero is generally avoided (causes symmetry problems), biases are often initialized to zero.

‚ö†Ô∏è Important Note:

Initializing all weights to zero in a neural network causes all neurons to learn the same features, preventing the network from learning effectively. This is why random initialization is preferred.

3. Zero-Mean Normalization

Data preprocessing often involves centering data around zero to improve training stability. 

Original Data: [10, 20, 30, 40, 50]

‚Ä¢ Faster convergence
